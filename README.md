# Квантовый генератор случайных чисел (КГСЧ)

## Описание системы

Данный проект представляет собой генератор истинно случайных чисел (TRNG). Система сочетает в себе оптимизированный аналоговый тракт с синхронной модуляцией источника энтропии и легковесный криптографический экстрактор на базе ARX-операций с межблочным связыванием.

## Физический источник энтропии

### Принцип действия: Фотоэффект
Источником случайности в данной системе является вероятностная природа фотоэффекта. При падении фотона на поверхность фотодиода акт выбивания электрона является истинно случайным квантовым событием. 

Даже при идеально стабильном световом потоке моменты регистрации отдельных фотонов и последующая генерация носителей заряда распределены во времени неравномерно. Эта фундаментальная квантовая неопределенность процесса взаимодействия света с веществом создает стохастическую компоненту тока, которая не может быть предсказана или воспроизведена детерминированными алгоритмами.

### Синхронная импульсная модуляция
Для выделения квантовой компоненты фотоэффекта и подавления классических шумов применяется метод стробирования сигнала:

1.  ИК-светодиод управляется короткими импульсами длительностью ~5 мкс непосредственно перед каждым измерением АЦП.
2.  Аналого-цифровой преобразователь выполняет выборку только в момент активной фазы вспышки, фиксируя мгновенный отклик фотодиода на квантовые акты фотоэффекта.
3.  В паузах между импульсами система находится в состоянии покоя, что разрывает временную корреляцию между соседними отсчетами и устраняет влияние тепловых и сетевых дрейфов.

#### Аналоговый тракт
Сигнал с фотодиода преобразуется в напряжение с помощью операционного усилителя в режиме трансимпедансного усилителя. Полосы пропускания и подавления шумов включает:

1.  Резистор обратной связи: Вместе с паразитной емкостью входа ОУ и фотодиода он образует интегрирующую цепь, ограничивающую полосу пропускания и подавляющую высокочастотные шумы.
2.  Декуплирующие конденсаторы: Пара конденсаторов (керамический и электролитический), подключенных к выводам питания ОУ, сглаживают высокочастотные помехи и предотвращает самовозбуждение усилителя за счет локального накопления заряда.

Дополнительно применяется оптическая изоляция пары «светодиод–фотодиод» для исключения паразитной засветки.

Оцифровка выполняется встроенным 12-битным АЦП микроконтроллера Waveshare RP2040 One с тактовой частотой дискретизации ~250 кГц. Передача данных на хост осуществляется по USB со скоростью 2 Мбит/с в пакетном режиме.

## Алгоритм экстракции энтропии

### Экстрактор ARX
Для преобразования сырого потока с неполной энтропией в равномерно распределенную бинарную последовательность используется блочный экстрактор на базе операций Add-Rotate-XOR (ARX).

Ключевые особенности реализации:

-   Предобработка сигнала: Перед подачей в ARX-сеть сырые данные с АЦП подвергаются процедуре декорреляции:
    1. Извлечение битовых плоскостей (младший, 4-й и 6-й биты) для захвата шума из разных разрядов оцифровки
    2. XOR этих плоскостей для усреднения шумовых компонент
    3. Дифференцирование полученной последовательности для устранения остаточного постоянного смещения
    4. Взятие младшего бита разности в качестве входа для экстрактора
-   Межблочное связывание: Состояние внутреннего регистра после обработки предыдущего блока используется как вектор инициализации для следующего. Это гарантирует, что выходная последовательность зависит от всей истории входных данных, устраняя долгосрочные корреляции физического источника.
-   24-х раундовое перемешивание: Каждый блок проходит через серию раундов, включающих XOR с раундовыми константами, циклические сдвиги на фиксированные величины и линейное смешивание слов состояния. Полная диффузия достигается за счет каскадного распространения изменений по всем битам состояния.
-   Иррациональные константы: Раундовые константы получены из дробных частей математических констант $\phi$, $\sqrt{2}$, $\pi$, $e$, $\sqrt{3}$, $\ln 2$, Апери, Эйлера-Маскерони, Каталана, Ландау-Рамануджана, Фейгенбаума α и δ. Использование трансцендентных и иррациональных чисел гарантирует, что их двоичное представление не содержит периодических паттернов.

### Альтернативные методы экстракции
Помимо собственного ARX-экстрактора, система поддерживает использование стандартизированных криптографических хеш-функций SHA2, SHA3, BLAKE, Streebog в режиме блочного хеширования с прореживанием выхода.

## Оценка качества источника


Оценка минимальной энтропии физического источника выполняется в соответствии с NIST SP 800-90B. Стандарт определяет набор из 10 оценщиков, каждый из которых выявляет специфический тип дефектов или зависимостей в источнике случайности. Итоговая оценка принимается как минимум из всех полученных значений, что обеспечивает консервативный результат.

### Реализованные оценщики

| Оценщик | Описание |
| :--- | :--- |
| Most Common Value (MCV) | Оценка по верхней границе доверительного интервала для вероятности наиболее частого символа |
| Collision Estimate | Оценка на основе среднего времени до первого повторения значения (метод Hagerty-Draper) |
| Markov Estimate | Оценка через модель Маркова 1-го порядка; учитывает зависимости между соседними символами |
| Compression Estimate | Оценка на основе статистики Маурера (LZ78); измеряет сжимаемость данных без предположения о независимости |
| t-Tuple Estimate | Обобщение MCV для t-грамм; выявляет зависимости в коротких последовательностях длиной $t$ |
| Longest Repeated Substring (LRS) | Оценка на основе длины самой длинной повторяющейся подстроки |
| Multi-MCW | Предсказание следующего значения через несколько окон «самого частого значения» (Multi Most Common in Window) |
| Lag Prediction | Предсказание на основе различных временных лагов (до 128 позиций назад) |
| MultiMMC | Марковские модели с подсчетом переходов разных порядков (Multi Markov Model with Counting) |
| LZ78Y Prediction | Предсказание на основе словаря LZ78 с модификацией Yabba (Bernstein's scheme) |

> Примечание: В текущей реализации Longest Repeated Substring отключен из-за нестабильности алгоритма при работе с зашумленными данными АЦП, и LZ78Y Prediction - из-за слишком большой длительности оценки

## Методология статистической верификации

### Тесты NIST SP 800-22
Базовый набор из 15 статистических тестов, рекомендованный Национальным институтом стандартов США для оценки генераторов случайных чисел. Данные тесты проверяют гипотезу о том, что битовая последовательность является независимой и равномерно распределенной. Включают проверку частотного баланса, длин серий, рангов бинарных матриц, спектральных характеристик и универсальной энтропии. Прохождение данного набора является необходимым условием применимости генератора в криптографических приложениях.

### Дополнительные метрики качества
Помимо стандартного набора NIST, применяются специализированные тесты для выявления скрытых артефактов экстракции:

-   Chi-Square Byte Test: Проверяет равномерность распределения значений байтов в выходном потоке. Позволяет обнаружить систематические перекосы, возникающие при некорректной группировке битов или остаточных зависимостях в младших разрядах АЦП, которые не фиксируются побитовыми тестами NIST.
-   Autocorrelation Test: Анализирует линейную зависимость между битами на различных временных лагах. В отличие от спектрального теста NIST, чувствительного к сильным периодическим компонентам, ACF выявляет слабые локальные корреляции.

### Критерии приемки
Успешное прохождение всех 17 тестов на уровне значимости α = 0.01
